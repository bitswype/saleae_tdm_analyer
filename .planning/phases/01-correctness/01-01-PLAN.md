  <action>
Convert all sprintf buffer-chaining patterns to snprintf with a tracked `used` offset variable. There are three locations with the `error_str[80]` + `warning_str[32]` pattern, plus additional sprintf calls that must be converted before pragmas can be removed.

**Location 1 — src/TdmAnalyzer.cpp (AnalyzeTdmSlot, ~lines 311-329):**
Replace the chained `sprintf(error_str + strlen(error_str), ...)` pattern with:
```cpp
char error_str[80] = "";
size_t used = 0;
if(frame.mFlags & SHORT_SLOT) {
    used += snprintf(error_str + used, sizeof(error_str) - used, "E: Short Slot ");
}
if(frame.mFlags & MISSED_DATA) {
    used += snprintf(error_str + used, sizeof(error_str) - used, "E: Data Error ");
}
if(frame.mFlags & MISSED_FRAME_SYNC) {
    used += snprintf(error_str + used, sizeof(error_str) - used, "E: Frame Sync Missed ");
}
if(frame.mFlags & BITCLOCK_ERROR) {
    used += snprintf(error_str + used, sizeof(error_str) - used, "E: Bitclock Error ");
}
```
Also convert the `warning_str[32]` sprintf to snprintf for consistency (the UNEXPECTED_BITS / Extra Slot string).

**Location 2 — src/TdmAnalyzerResults.cpp (GenerateBubbleText, ~lines 51-71):**
Apply the same snprintf + used offset pattern to `error_str[80]` and `warning_str[32]`. Also check `channel_num_str[128]` — if it uses sprintf, convert to snprintf for pragma removal.

**Location 3 — src/TdmAnalyzerResults.cpp (GenerateFrameTabularText, ~lines 490-510):**
Apply the same snprintf + used offset pattern to `error_str[80]` and `warning_str[32]`.

**Pragma removal (LAST — after all call sites converted):**

- Remove `#pragma warning( disable : 4996 )` from `src/TdmAnalyzerSettings.cpp` (~line 8)

**IMPORTANT: Do NOT remove pragmas before all sprintf calls in those files are converted.** The build will break on MSVC with C4996 errors.

**CORR-03 — src/TdmAnalyzerResults.cpp (GenerateWAV, ~lines 176-193):**
Inside the `for( U64 i = 0; i < num_frames; i++ )` loop, within the `if(frame.mType < num_slots_per_frame)` block, change:
```cpp
wave_file_handler.addSample( frame.mData1 );
```
to:
```cpp
// SHORT_SLOT frames contribute silence (0) to preserve WAV channel alignment.
// Skipping addSample() entirely would cause all subsequent channels to drift
// by one slot position in the output file.
U64 sample_value = (frame.mFlags & SHORT_SLOT) ? 0 : frame.mData1;
wave_file_handler.addSample( sample_value );
```
The condition `(frame.mFlags & SHORT_SLOT)` MUST be inside the existing `if(frame.mType < num_slots_per_frame)` guard. Do NOT add a separate outer condition.

**CORR-04 — src/TdmAnalyzerResults.cpp (GenerateFrameTabularText, ~line 471):**

```cpp
// Required by AnalyzerSDK >= 1.1.32: ClearTabularText() must be the first
// executable call in GenerateFrameTabularText(). Omitting it causes Logic 2
// to crash when displaying tabular data. Do not move or remove this call.
ClearTabularText();
```
This is a comment-only change. Do NOT add a second ClearTabularText() call — one already exists.
