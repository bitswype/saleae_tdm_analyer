  author        — author name
  description   — human-readable description

entryPoint resolution rule:
  File: hla-wav-export/TdmWavExport.py
  Class: TdmWavExport
  entryPoint value: "TdmWavExport.TdmWavExport"

saleae.analyzers imports needed:
  HighLevelAnalyzer, AnalyzerFrame, StringSetting, ChoicesSetting

Settings API rules:
  - Declared as class-level attributes (NOT in __init__)
  - Logic 2 injects values as instance attributes before __init__ runs
  - ChoicesSetting(choices, **kwargs): choices is FIRST positional arg
  - .default attribute set post-construction: bit_depth.default = '16'
  - No default= kwarg in ChoicesSetting constructor (raises TypeError)
  - StringSetting(label=...) for free-form text inputs

FrameV2 schema from TdmAnalyzer.cpp:
  Normal frame: frame.type == "slot"
    frame.data["slot"]              -> int, 0-based slot index
    frame.data["data"]              -> int, signed-adjusted sample
    frame.data["frame_number"]      -> int
    frame.data["severity"]          -> str: "ok"/"warning"/"error"
    frame.data["short_slot"]        -> bool
    frame.data["extra_slot"]        -> bool
    frame.data["bitclock_error"]    -> bool
    frame.data["missed_data"]       -> bool
    frame.data["missed_frame_sync"] -> bool
    frame.data["low_sample_rate"]   -> bool

  Advisory frame: frame.type == "advisory"
    frame.data["severity"]  -> "warning"
    frame.data["message"]   -> str

result_types format string syntax: double-brace, e.g. '{{data.message}}'
