After line 65 (the `#pragma scalar_storage_order default` following WavePCMHeader), insert:

```cpp

// Verify WavePCMHeader is the correct size for WAV PCM format.
// Note: #pragma scalar_storage_order is GCC-only and ignored by Clang/MSVC.
// This assert catches packing errors on all supported compilers.
static_assert(sizeof(WavePCMHeader) == 44,
    "WavePCMHeader must be 44 bytes per WAV PCM spec. "
    "Check #pragma pack(1) is in effect and all field types are correct.");
```

After line 94 (the `#pragma scalar_storage_order default` following WavePCMExtendedHeader), insert:

```cpp

// Verify WavePCMExtendedHeader is the correct size for WAV EXTENSIBLE format.
// Note: #pragma scalar_storage_order is GCC-only and ignored by Clang/MSVC.
// This assert catches packing errors on all supported compilers.
static_assert(sizeof(WavePCMExtendedHeader) == 80,
    "WavePCMExtendedHeader must be 80 bytes per WAV EXTENSIBLE spec. "
    "Check #pragma pack(1) is in effect and all field types are correct.");
```

Placement rule: Both static_assert statements must appear AFTER `#pragma pack(pop)` and `#pragma scalar_storage_order default` — fully outside both pragma regions.

Do NOT remove or modify `#pragma scalar_storage_order` — it documents intent and provides enforcement on GCC.
Do NOT place static_assert inside the struct body or inside the pragma pack region.
Do NOT swap the expected sizes — WavePCMHeader is 44, WavePCMExtendedHeader is 80. Cross-check against the `/* data */` offset comments in each struct.
