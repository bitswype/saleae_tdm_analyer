### Integration Observation

Advisory FrameV2 ("advisory" type) does not carry the `low_sample_rate` field — only "slot" type frames do. HLA scripts must check `frame.type == "slot"` before accessing slot-only fields. This is documented in CHANGELOG migration section and is by design, not a defect.
